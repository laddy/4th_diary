# 4th_diary

* PHP 5.3 or greater
* MySQL 5 or greater

## Install

Use SlimFramework

- http://www.slimframework.com/

### composer install

```
$ curl -s https://getcomposer.org/installer | php
```

### composer.json

```
{
    "require": {
        "slim/slim": "2.*"
    }
}
```

### Slim Install

```
$ ./composer.phar install
```

## Run

### PHP Build In Web Server

```
$ php -S localhost:9000
```
